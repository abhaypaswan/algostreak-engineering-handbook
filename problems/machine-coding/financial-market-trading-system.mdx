You are tasked with designing a control system for a financial market trading system. The securities can be bought and sold. The system should keep track of the prices, orders, and transactions. The system should also have an option to handle the emergency situations like market crash and the system should also be able to handle multiple markets at the same time. The system should also have an interface for the management to monitor the market's status and usage.
